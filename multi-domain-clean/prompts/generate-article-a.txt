QUALITY RULES (AVOID BAD OUTPUTS)
---------------------------------------

Why I Love This Recipe lightbulb:
- Write EXACTLY 4 items. Each item must be DIFFERENT (no repetition).
- Use a REAL short label in <strong>...</strong>, e.g. <strong>Quick Prep</strong>, <strong>One-Pot Convenience</strong>. NEVER use placeholder words like "BoldLabel".
- Avoid generic repetition (e.g. "perfect for busy nights" in every item). Vary the reasons.

Pro Tips lightbulb:
- Write EXACTLY 4 separate items. Each tip must be a distinct pro-tip item, not a numbered list crammed into one item.
- One concise tip per item. Do NOT put multiple tips with "**1.**", "**2.**" inside a single item.

FAQs section:
- Write 2–4 REAL questions that readers actually ask (e.g. "Can I use frozen chicken?", "How long does this keep in the fridge?").
- Write REAL answers (2–4 sentences each). NEVER write "Please see the recipe above" or "Question about X? Answer about X." as placeholders.
- Each FAQ must add value; do not duplicate the same question.

Step-by-Step Instructions:
- Each step must have ACTUAL content. NEVER leave steps empty.
- Write specific, actionable instructions for each step.
- Phase-based H3 sections must each contain a full paragraph of instructions.

Paragraphs:
- Keep paragraphs readable: 2–5 sentences each. Avoid walls of text (200+ word single paragraphs).
- Ingredients paragraph: be concise. Do not over-explain every single ingredient in one block.

Variations / bullet lists:
- Use simple format: one variation per item. Do NOT put table rows (| Option | Description |) inside list items.
- Bullet items: plain text or <strong>Label</strong> text. No markdown, no pipes.
